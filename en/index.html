About Alis

On A Strong Foundation

Alis LGX is based on Arch Linux.
Arch is a reasonably popular distro due to its simplicity of structure, freshness of software and its unmatched robustness, and Alis matches most of these characteristics.

Extensive Softwares

Alis borrows many software packages from Arch's repository, but software not listed there can be introduced from the Alis' repository.
In addition, the Chaotic AUR repository, where pre-built AUR packages are placed, is enabled as standard, reducing the need to build AUR packages on your own. Furthermore, the graphical software centres 'Discover' (LXQt, Plasma) and 'GNOME Software' (Xfce, GNOME) are included as standard. There is no need to go through the hassle of command operations to install softwares.

Optimisation That You Could Cringe

Alis' standard kernel is Linux Realtime, which has been optimised to be preemptive and low-latency through special patches.
Furthermore, the desktop environment, which is known to be heavy, has been lightened by adjusting the daemons that start up. Once you use this, you may never go back to the OS used so far.

Easy Installation

Arch comes with a text-based installer, but it is difficult for beginners to operate.
Alis has a 'Calamares' installer. Time zones can be selected on a map, the keyboard layout is detailed, and fixed drive partitioning is easy with mouse and keyboard.

Semi-permanent Use

Alis is a rolling release. There are no major updates every few months or years.
The OS only needs to be installed once, and there is no need to download the ISO again for updates!
