フレーバーの特徴

こんな感じでどうですかね

LXQt

質実剛健を体現したようなデスクトップ環境。軽いのにデスクトップとしての機能は一通り揃っている。

スペックの目安

CPU 1GHz以上のx86_64
RAM 512 MB(1 GB推奨)
固定ドライブ 20 GB以上

LXQt Amber

比較的古い（2～8世代辺り）インテルグラフィックスを搭載した機種向け。いくら新しくとも、歴史に目を背けるコトはないのだ。

スペックの目安

CPU 1GHz以上のx86_64
RAM 512 MB(1 GB推奨)
固定ドライブ 20 GB以上

GNOME

Win、Macどちらの操作体系にも似ない。見た目より軽いかも。

スペックの目安

CPU 1GHz以上のx86_64
RAM 768 GB(2 GB推奨)
固定ドライブ 20 GB以上

Plasma

多分最も近代的なフレーバー。タスクバーが浮く、ウインドウがゼリーみたいにプルプルするなどの画面効果もあるよ。

スペックの目安

CPU 1GHz以上のx86_64
RAM 768 MB(2 GB推奨)
固定ドライブ 20 GB以上

Xfce

その豪華な見た目に反して軽量。カフェでドヤれそうな見た目だけどあんまり意味はないかも。

スペックの目安

CPU 1GHz以上のx86_64
RAM 640 MB(1.2 GB推奨)
固定ドライブ 20 GB以上

有償版を購入

有償版は以下の2種類を予定しています。価格はどちらも2,200円（税込み）です。

  • Professional …GNOME版に「Multivalent」及び「Chaotic AUR」から厳選したソフトウェアを同梱したエディション
  • Studio …Plasma版をマルチメディア編集に特化させたエディション

どちらのエディションにも、メールによるユーザー登録から30日間のインストールサポートが付属します。

各種同人誌即売会、及びNJBの公式通販にて販売予定です。

既存環境からの移行

以下の手順で移行を行なって下さい。

sudo pacman -Syu
wget https://alislinux.github.io/migrate2alis.sh
sudo bash migrate2alis.sh

ライセンス

スクリプト実行時にライセンスが表示されるので、それをご覧下さい。
尚、スクリプト自体は三条項BSDライセンスの下で許諾されます。

対応しないOS

  • Manjaro及び派生ディストロははArchのリポジトリを間借りしていないため、移行できません。
  • initシステムにsystemdを用いないシステムからの移行はできません。
  • KaOS、Frugalware LinuxはPacmanを採用しながら、Archのシステムに基づいていないため、Alisへの移行はできません。
  • その他Arch派生でないディストリビューションからの移行はできません。新規にインストールして下さい。
